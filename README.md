# Camera Monitoring Suite AI/CV

A comprehensive system for AI-powered camera monitoring and computer vision.

## Project Structure
- `TitSmartDashboardBackend`: Node.js/Express backend service.
- `TitSmartDashboardWeb`: React/Vite frontend application.

## Prerequisites
- Node.js (v20+ recommended)
- npm or pnpm

## Setup and Running

### 1. Backend Service
```bash
cd TitSmartDashboardBackend
npm install
# Create .env file based on your environment
npm run dev
```

### 2. Web Application
```bash
cd TitSmartDashboardWeb
npm install
npm run dev
```

## Features
- AI-based camera monitoring.
- Real-time dashboard interface.
- Scalable backend architecture.
